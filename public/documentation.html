MyFlix API

This is a Node.js and Express application. This API uses REST architecture, with URL endpoints corresponding to data operations of movie information in JSON format using standard HTTP response codes


Business Logic Endpoint URL HTTP Method Reqest Body Response Data
Get All Movies /movies GET None JSON array of movie objects containing description, genre, director, image URL, and featured status.
Get Movie by Title /movies[title] GET None A JSON object holding data about a single movie, containing a description, genre, director, image URL property. Example:
{
title: "Titanic",
genre: "Romance",
director: "James Cameron",
"image URL": "https://en.wikipedia.org/wiki/Titanic_(1997_film)#/media/File:Titanic_(1997_film)_poster.png"
}
Get Movie by Genre /movies[genre] GET None A JSON object holding data about a name/title by genre (description)
Get Director Information by Title or Director /movies[director] GET None A JSON object holding data about a director of a specific movie. Example:
{
director: "James Cameron",
bio: "is a Canadian filmmaker. He is a major figure in the post-New Hollywood era. He often uses novel technologies with a classical filmmaking style",
"birth year": "1954",
"death year": "Living"
}
New User Register /users POST A JSON object holding data about the user to add. Example:

{
Name: 'John Doe',
Username: johndoe101,
Email: johndoe@email.com,
'Birth-date': 'YYYY-MM-DD'
} 		A JSON object holding data about the user that was added, including their assigned id and auth token:
{
'_id': '######',
Name: 'John Doe',
Username: johndoe101,
Email: johndoe@email.com,
'Birth-date: 'YYYY-MM-DD',
token: '######'
}
Update User Information /users[profile] PUT A JSON object holding data about the user to update. Example:

{
Name: 'John Doe',
Username: johndoe101,
Email: johndoe@email.com,
'Birth-date': 'YYYY-MM-DD'
} 		Allow users to update their user information. Example:
{ '_id': '######',
Name: 'John Doe',
Username: johndoe101,
Email: johndoe@email.com,
'Birth-date: 'YYYY-MM-DD',
token: '######' }
Movie Favorite List /users[profile]/movies[title] POST None Allow users to add a movie to their list of favorites with a text message indicating whether the movie was successfully added from the user's favorited movies
Remove from Favorite List /users[profile]/movies[title] DELETE None Allow users to remove a movie from their list of favorites with a text message indicating whether the movie was successfully removed from the user's favorited movies
Deregister User /users[profile] DELETE None Allow existing users to deregister with a confirmation stating the user has been deleted
